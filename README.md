# CrypticCoin Source Code

## Specifications

* PoW (proof of work)
* Algorithms: scrypt, x17, Lyra2rev2, myr-groestl, & blake2s
* Blocktime: 30 seconds
* Total Supply: 7.6 Billion
* 55% pre-mined
* FreeCO
* RPC port: `23202`
* P2P port: `23303`
